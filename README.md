## ✨ 功能介绍

本工具可以帮助你：

- 🎯 **自动识别开局词条** - 自动检测「货币战争」模式中的 Boss 词条、投资环境、投资策略
- 🔄 **自动重开** - 不符合期望开局时自动重新开始
- 📱 **手机自动化** - 通过 ADB 连接安卓手机，自动点击操作
- 🖼️ **截图保存** - 可选保存检测到的开局界面图片
- 🔧 **调试模式** - 可选保存操作过程截图，方便排查问题

### 支持的开局检测

| 类型 | 说明 |
|------|------|
| **Boss 词条** | 变宝为废、轮回不止等 |
| **投资环境** | 彩虹时代、头彩、特权阶级、夜之半神邀请等 |
| **投资策略** | 快请专家、摸个鱼吧、加拉赫、停云、青雀等 |

### 战斗流程说明

本工具的战斗逻辑如下：

1. **自动拖人** - 将 4 个角色从后台拖到前台
2. **自动战斗** - 依赖游戏自带的「自动战斗」功能（需要提前在游戏设置中开启「沿用自动战斗设置」）
3. **刷取范围** - 目前只刷到**第一面（1-3）的投资策略选择**为止

⚠️ **注意：** 第二面及后续的战斗策略暂未支持，后续版本可能会添加。

## 📦 使用前准备

### 1. 确认手机分辨率

本工具要求手机分辨率为 **1080x2400**（竖屏），有两种方法可以确认：

#### 方法一：使用 ADB 命令（推荐，最准确）

1. 打开 `srcwroller` 文件夹
2. 在空白处右键，选择「在终端中打开」
3. 输入以下命令：

```bash
.\adb shell wm size
```

4. 如果输出 `Physical size: 1080x2400`，说明分辨率符合要求

#### 方法二：手机设置查看

```
设置 → 显示 → 屏幕分辨率
```

⚠️ **注意：** 如果分辨率不是 1080x2400，工具可能无法正确识别界面元素，不建议使用！

### 2. 开启手机 USB 调试

1. 打开手机的「设置」→「关于手机」
2. 连续点击「版本号」7 次，开启「开发者选项」
3. 返回设置，找到「开发者选项」
4. 开启「USB 调试」

### 3. 连接手机到电脑

1. 使用数据线将手机连接到电脑
2. 手机上会弹出「允许 USB 调试」的提示，点击「允许」

## 🚀 快速开始（小白必看）

### 步骤 1：下载并解压

从 GitHub Releases 页面下载最新版本的发布包，解压到任意目录。

解压后的目录结构如下：

```
SRCWRoller/
├── srcwroller/
│   ├── srcwroller.exe      # 主程序
│   ├── adb.exe             # ADB 工具（已内置）
│   ├── AdbWinApi.dll       # ADB 依赖文件
│   ├── AdbWinUsbApi.dll
│   ├── config.yaml         # 配置文件
│   └── models/             # OCR 模型文件
│       ├── ch_PP-OCRv4_det_infer.mnn
│       ├── ch_PP-OCRv4_rec_infer.mnn
│       └── ppocr_keys_v4.txt
├── config.yaml             # 配置文件（可复制使用）
└── README.md               # 本说明文档
```

### 步骤 2：游戏设置（重要）

在开始之前，需要确保游戏已开启自动战斗功能：

1. 打开游戏，进入「设置」→「战斗功能」
2. 确保**「沿用自动战斗设置」**已开启（打开开关）
3. 这样每次进入战斗会自动进行，无需手动操作

⚠️ **如果不打开这个设置，工具无法正常运行！**

### 步骤 3：确认启动页面

**非常重要：** 运行工具前，必须手动将游戏进入到一个特定的起始页面。

请按照以下步骤操作：

1. 打开《崩坏：星穹铁道》游戏
2. 进入「货币战争」活动界面
3. 确保屏幕显示如下图类似的画面（底部有「开始『货币战争』」按钮）：

![起始页面示例](assets/startpage.png)

**必须在这个页面才能开始运行工具！**

### 步骤 4：修改配置文件

编辑 `srcwroller/config.yaml` 文件。参考文件里面的配置和下面的配置文件说明来设置自己要刷的开局。

配置文件详解请看下面的 [配置文件说明](#-配置文件说明) 章节。

### 步骤 4：运行程序

#### 方法一：双击运行（推荐）

直接双击 `srcwroller/srcwroller.exe` 即可开始运行。

#### 方法二：在终端中打开（推荐）

1. 打开 `srcwroller` 文件夹
2. 在空白处右键，选择「在终端中打开」（Windows 11）或「在此处打开 PowerShell 窗口」（Windows 10）
3. 输入以下命令：

```bash
.\srcwroller.exe
```

#### 方法三：命令行运行

1. 按 `Win + R`，输入 `cmd`，按回车
2. 输入以下命令（将路径替换为你的实际路径）：

```bash
cd /d D:\你的下载路径\SRCWRoller\srcwroller
srcwroller.exe
```

### 步骤 5：等待运行完成

程序开始运行后会自动：

1. 检测当前是否在起始页面
2. 点击「开始货币战争」
3. 识别 Boss 词条 → 点击「下一步」
4. 点击「开始对局」→ 点击「点击空白处继续」
5. 识别投资环境 → 根据配置选择或刷新
6. 进入准备阶段 → 自动将 4 个角色拖上前台
7. 进入战斗 → **依靠游戏自动战斗**（需要提前开启自动战斗）
8. 战斗胜利 → 点击「继续挑战」
9. 识别投资策略 → 根据配置选择或刷新
10. 判断是否符合期望开局 → 不符合则重开，符合则保留

⚠️ **注意：** 工具目前只刷到**第一面（1-3）的投资策略选择**为止。后续的战斗和第二面暂时不支持自动处理。

你可以在命令行窗口看到运行日志。

## ⚙️ 配置文件说明

配置文件 `config.yaml` 用于设置你期望的开局条件。

### 配置文件位置

- **推荐：** 放在 `srcwroller/config.yaml`（和程序同一目录）
- **或者：** 放在当前运行目录

### 配置文件内容详解

```yaml
# 最大重试次数（默认 500 次）
max_retry: 500

# 优先的投资环境（满足这些环境之一就会保留）
prefer_env:
  - 彩虹时代
  - 头彩
  - 特权阶级
  - 夜之半神邀请
  - 银河学者邀请
  - 银·金·彩

# 开局配置列表（可以配置多组）
openings:
  # 第一组配置
  - env: []  # 环境为空（任意环境都可以）
    strategy: ["轮回不止"]  # 必须包含「轮回不止」策略
    affix: ["变宝为废"]     # 必须包含「变宝为废」Boss 词条

  # 第二组配置（示例，可以删除或修改）
  - env:
      - 专家研讨会
      - 特邀专家
    strategy:
      - 快请专家
      - 摸个鱼吧
      - 加拉赫
      - 停云
      - 青雀
    affix: []  # Boss 词条为空（任意词条都可以）

# 设备序列号（一般不需要填，自动检测）
device_serial: null

# OCR 识别置信度（默认 0.7，不要修改）
confidence: 0.7

# 点击间隔时间（秒，默认 0.5）
click_sleep: 0.5

# 页面检测超时时间（秒，默认 10）
page_timeout: 10

# 调试模式（默认 false，开启后会输出更多日志）
debug: false

# OCR 模型配置（一般不需要修改）
ocr:
  models_dir: ./models
  det_model: ch_PP-OCRv4_det_infer.mnn
  rec_model: ch_PP-OCRv4_rec_infer.mnn
  keys_file: ppocr_keys_v4.txt
```

### 配置示例

#### 示例 1：只要「轮回不止 + 变宝为废」开局

```yaml
max_retry: 500
prefer_env: []
openings:
  - env: []
    strategy: ["轮回不止"]
    affix: ["变宝为废"]
```

#### 示例 2：要特定环境和策略

```yaml
max_retry: 500
prefer_env: ["彩虹时代", "头彩"]
openings:
  - env: ["彩虹时代"]
    strategy: ["快请专家", "摸个鱼吧"]
    affix: []
```

#### 示例 3：任意开局都接受（相当于不刷）

```yaml
max_retry: 1
prefer_env: []
openings:
  - env: []
    strategy: []
    affix: []
```

## ❓ 常见问题

### Q1: 程序提示「ADB 连接失败」

**解决方法：**

1. 确保手机已开启 USB 调试
2. 确保手机通过数据线连接到电脑
3. 检查手机上是否弹出了「允许 USB 调试」的提示
4. 尝试重新插拔数据线
5. 如果是第一次连接，可能需要在电脑上安装手机驱动程序

### Q2: 程序一直说「未找到起始页面」

**解决方法：**

1. 确认游戏已经进入到「货币战争」活动的起始页面
2. 确认手机分辨率是 1080x2400
3. 确认屏幕没有被其他通知、弹窗遮挡
4. 可以尝试手动调整一下游戏窗口位置

### Q3: 识别不准确/总是识别错误

**解决方法：**

1. 确保手机分辨率是 1080x2400
2. 检查游戏画质设置，确保文字清晰
3. 可以尝试调整 `confidence` 参数（降低到 0.6）
4. 开启 `debug: true` 查看详细日志

### Q4: 点击位置不对

**可能原因：**

- 手机分辨率不是 1080x2400
- 游戏画面比例不对

**解决方法：** 本工具仅支持 1080x2400 分辨率（竖屏），其他分辨率无法保证效果。

### Q5: 如何停止程序？

按 `Ctrl + C` 可以强制停止程序运行。

### Q6: 会不会被封号？

本工具仅模拟点击操作，不修改游戏数据，理论上风险较低。但使用任何第三方工具都有潜在风险，请自行评估。

## 🔧 高级功能

### 调试模式

如果你遇到了问题，可以开启调试模式获取更多信息：

```bash
srcwroller.exe --debug true
```

### 保存操作截图

运行程序时添加参数可以保存每次点击的截图：

```bash
srcwroller.exe --debug_ops debug_ops
```

截图会保存在 `debug_ops` 文件夹中。

### 保存开局界面

如果你想查看检测到的开局界面，可以开启保存功能：

```bash
srcwroller.exe --save_opening opening_screens
```

开局截图会保存在 `opening_screens` 文件夹中，文件名为：
- `boss_affix_时间戳.png` - Boss 词条页面
- `invest_env_时间戳.png` - 投资环境页面
- `invest_strategy_时间戳.png` - 投资策略页面

### 测试页面功能

如果你想测试某个页面识别是否正常：

```bash
# 测试所有页面识别
srcwroller.exe test-page

# 测试特定页面
srcwroller.exe test-page --page StartPage

# 使用截图文件测试（不需要连接手机）
srcwroller.exe test-page --screenshot test.png
```

### 创建示例配置文件

```bash
srcwroller.exe --example
```

会创建一个 `config.example.yaml` 示例配置文件。
